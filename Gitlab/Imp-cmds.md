# IMPORTANT CMDS

## Cloning repo
```
 git clone https://oauth2:<gitlab token>@<git lab ssh/ http url>
```

## Logs
```
git logs 
```

## switching branch
```
git checkout <branch name>
```

## cheaking branches
```
git branch
```

## Rollback
```
git reset --hard <commit id>
```

## cheking the difference 
```
git diff <old commit id>..<new commit id>
```
